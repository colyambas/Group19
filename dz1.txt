Вот сценарий который надо уметь делать

1) Посмотреть где я			pwd
2) Создать папку			mkdir folder1
3) Зайти в папку			cd folder1
4) Создать 3 папки			mkdir {folder1.1,folder1.2,folder1.3}
5) Зайти в любоую папку		cd folder1.1
6) Создать 5 файлов (3 txt, 2 json)	touch 1.txt 2.txt 3.txt 1.json 2.json
7) Создать 3 папки			mkdir {!,+,+++}
8. Вывести список содержимого папки	ls -al
9) + Открыть любой txt файл		nano 1.txt
10) + написать туда что-нибудь, любой текст.
11) + сохранить и выйти. 		Ctrl+x
12) Выйти из папки на уровень выше	cd ..

13) переместить любые 2 файла, которые вы создали, в любую другую папку.
	mv folder1.1/{1.txt,2.txt} folder1.2
14) скопировать любые 2 файла, которые вы создали, в любую другую папку.
	cp {1.json,2.json} ..
15) Найти файл по имени
	cd ~
	find . -depth -name “1.txt”
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает.
	grep -iwr “перемен” ./*
17) вывести несколько первых строк из текстового файла
	head -5 1.txt
18) вывести несколько последних строк из текстового файла
	tail -5 1.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает.
